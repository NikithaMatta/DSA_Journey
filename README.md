# DSA-Journey-LeetCode

LeetCode Solution Repository!

This repository is dedicated to tracking progress and mastery of Data Structures and Algorithms (DSA). <br>
It contains solutions to LeetCode problems, organized by topic.

## Repository Structure

Problems are categorized by topics.

```scss
Arrays
Strings
.
.
.
```

More topics will be added as the learning journey progresses, and solutions will be organized accordingly.

### How to Use This Repository

1. **Daily Commits:** Solutions to LeetCode problems are added regularly to maintain consistency.
2. **Topic-Wise Organization:** Solutions are organized by topic (Arrays, Strings, etc.) and further categorized by difficulty level (Easy, Medium, Hard).
3. **Solution Files:** Each solution is stored in its respective folder, with filenames indicating the problem name and difficulty level.

## Goals

* **Master DSA:** Build a strong foundation in Data Structures and Algorithms by consistently solving problems across topics such as Arrays, Strings, Dynamic Programming, and more.
* **Track Progress:** Document the journey from Easy to Hard problems while improving problem-solving skills.
* **Prepare for Interviews:** Develop the skills and confidence required for coding interviews through regular practice.

## Progress Tracking
